Budget Tracker - Phase 1 Complete
==================================

PROJECT LOCATION: /tmp/budget-tracker/

QUICK START
-----------
1. Open VSCode in the project directory
2. Run: npm install
3. Run: npm run dev
4. Open http://localhost:3000

DEFAULT LANDING PAGE
--------------------
Dashboard is set as the default route. App automatically redirects to Dashboard on load.

FEATURES IMPLEMENTED
--------------------
✓ Income & Expense Entry Form
  - Type selector (Income/Expense)
  - Category dropdown (Salary, Freelance, Housing, Food, etc.)
  - Expense type (Mandatory/Leisure)
  - Recurring toggle
  - Amount, Date, Notes fields

✓ Dashboard (Default Page)
  - Total Income metric
  - Total Expense metric
  - Money Saved (Income - Expense)
  - Daily Budget display
  - Line chart (Income vs Expense over time)
  - Recent transactions list

✓ Daily Budget Logic
  Formula: (Total Income - Total Mandatory Expenses) / Days in Current Month
  - Recalculates automatically on transaction changes
  - Only mandatory expenses deducted

✓ Calendar View
  - Monthly grid view
  - Daily income/expense summaries
  - Budget exceeded warnings
  - Mandatory vs Leisure indicators
  - Day detail panel with transactions

✓ Responsive Design
  - Mobile (360-430px): Single column
  - Tablet (768px): Adjusted layouts
  - Laptop (1024-1440px): Two-column grid
  - Desktop (1920px+): Max-width layout

✓ Logo
  - SVG located at: public/logo.svg
  - Nano Banana Prompt: See README.md
  - Blue gradient with green accent

✓ Architecture
  - Strict TypeScript types
  - Service layer for LocalStorage
  - No direct LocalStorage access in components
  - Modular component structure
  - Clean separation of concerns

ROUTES
------
/           → Dashboard (default)
/calendar   → Calendar View
*           → Redirects to /

CATEGORIES
----------
Income: Salary, Freelance, Bonus, Other
Expense: Housing, Food, Business, Transportation, Utilities, Other

TECH STACK
----------
- React 18 + TypeScript
- Vite (dev server + build)
- React Router
- Recharts (visualization)
- date-fns (date handling)
- Lucide React (icons)
- LocalStorage (persistence)

COMMANDS
--------
npm install       → Install dependencies
npm run dev       → Start dev server (port 3000)
npm run build     → Production build
npm run preview   → Preview production build
npm run lint      → Run ESLint
npm run typecheck → TypeScript check

DATA STORAGE
------------
All data persisted to browser LocalStorage
Key: 'budget-tracker-transactions'
Accessed ONLY through: src/services/storage.ts

RESPONSIVE BREAKPOINTS
----------------------

Tablet:    max-width: 768px   → Adjusted layouts
Laptop:    max-width: 1024px  → Two-column grid
Desktop:   min-width: 1920px  → Max-width 1600px
